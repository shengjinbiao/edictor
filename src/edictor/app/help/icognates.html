Compute Cognate Sets

This dialog lets you generate cognate IDs automatically.

  • THR. — clustering threshold for LexStat (LingPy); default 0.45.
  • Requirements — the wordlist must have DOCULECT, CONCEPT, and segmented TOKENS columns.
  • Backend — when the local Python server with LingPy is available, the LexStat method runs; otherwise the fallback consonant-class heuristic inside EDICTOR is used.
  • Output — results are written to the current cognate column (COGID/COGIDS); a column is created if missing.
